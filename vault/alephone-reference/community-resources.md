# Community Resources

Links to Marathon modding wikis, Aleph One documentation, scenario development resources, community forums, and tools.

## Official & Primary Resources

### Aleph One (Open Source Engine)

- **Aleph One Official Site**: [alephone.lhowon.org](https://alephone.lhowon.org/)
  The official home of Aleph One. Downloads for the engine, links to the three official scenarios (Marathon, Marathon 2, Marathon Infinity), and documentation.

- **Aleph One GitHub**: [github.com/Aleph-One-Marathon/alephone](https://github.com/Aleph-One-Marathon/alephone)
  The C++ source code repository. GPL-3.0 licensed.

- **Aleph One GitHub Wiki**: [github.com/Aleph-One-Marathon/alephone/wiki](https://github.com/Aleph-One-Marathon/alephone/wiki)
  Developer notes, file locations, build instructions, and technical documentation.

- **Aleph One Developer Notes**: [github.com/Aleph-One-Marathon/alephone/wiki/Developer-Notes](https://github.com/Aleph-One-Marathon/alephone/wiki/Developer-Notes)
  Film playback profiles, compatibility notes, and engine internals.

- **Marathon 2: Durandal**: The 1995 sequel

## Community Forums & Discussion

- Scenario development discussion
- Engine modification proposals
- Bug reports and technical help
- General Marathon community conversation
- Project announcements and showcases

## Content Repositories

### Simplici7y

- **URL**: [simplici7y.com](https://simplici7y.com/)
- The de facto file-sharing platform for the Marathon/Aleph One community
- Named after a "7-polygon mapping challenge" (doing more with less)
- Online since ~2007, over 12 million downloads, 600+ items, 1400+ reviews
- Categories: Scenarios, Maps, Physics mods, Shapes patches, Scripts, Utilities
- Originally Ruby on Rails, now Django 4 on Heroku
- GitHub: [github.com/rkuykendall/Simplici7y](https://github.com/rkuykendall/Simplici7y)

### Content Tags on Simplici7y

## Technical Documentation

### File Format Documentation

- **Marathon + Aleph One Format and Data Documentation**: [GitHub Gist by marrub--](https://gist.github.com/marrub--/98af41f36e15a277088b220a6a9f4244)
  The most comprehensive third-party documentation of Marathon's binary file formats. Covers WAD headers, map tags, shapes, sounds, and physics with byte-level detail.

- **Marathon Shapes File Format**: [dalcanton.it](https://www.dalcanton.it/tito/hp/marathonshapes/)
  Detailed documentation of the Shapes file format including collection headers, bitmaps, color tables, and compression.

- **MML Documentation**: [tst2005.github.io/alephone-doc/docs/MML.html](http://tst2005.github.io/alephone-doc/docs/MML.html)
  The Marathon Markup Language specification for Aleph One configuration.

### Map Physics Reference

- **Marathon 2 Physics Models**: [marathon.bungie.org/pmarchive/models2.htm](https://marathon.bungie.org/pmarchive/models2.htm)
  Reference for Marathon 2's physics constants and game model data.

## Development Tools

### ShapeFusion

- **GitHub**: [github.com/treellama/shapefusion](https://github.com/treellama/shapefusion)
- Open-source editor for Marathon Shapes, Sounds, and Physics files
- Can import/export bitmaps, edit color tables, modify physics constants
- Essential tool for Marathon content creation

### Forge (Level Editor)

The original level editor created by Bungie and shipped with Marathon Infinity. While archaic, it established the conventions for Marathon level design. Modern alternatives exist in the community.

### Anvil (Physics/Shapes Editor)

Bungie's original physics and shapes editor, also shipped with Marathon Infinity. ShapeFusion is the modern replacement.

### Marathon Utils

- **GitHub**: [github.com/Hopper262/marathon-utils](https://github.com/Hopper262/marathon-utils)
- Perl scripts for reading/writing Marathon Shapes and other data files
- Useful reference implementation for file format parsing

### Mapmaking Resources

- **Marathon Mapmaking Guide**: [aaronfreed.github.io/mapmaking101.html](https://aaronfreed.github.io/mapmaking101.html)
  Modern guide to Marathon level design: setup, common pitfalls, and solutions.

## Key Community Figures & Projects

The Marathon community, while small, has been continuously active since 1994. Notable community projects include:

- **Rubicon X** -- One of the most acclaimed third-party scenarios
- **Phoenix** -- Major community scenario
- **Eternal** -- Long-running community project
- Various "total conversions" that replace all game assets

The community is notable for its longevity (30+ years) and the depth of technical knowledge preserved about the engine internals.
